# @reino-doce/streaming-chat-webclient-cli

Read-only CLI for `@reino-doce/streaming-chat-webclient`.

Repository: https://github.com/Reino-Doce/streaming-chat-webclient/tree/main/packages/streaming-chat-webclient-cli

## Install

```bash
npm install @reino-doce/streaming-chat-webclient-cli
```

## Usage

```bash
streaming-chat-webclient connect --host 127.0.0.1 --port 5443 --token my-token
```

CLI help/output is intentionally English-only. 

## Options

- `--host <host>`
- `--port <port>`
- `--token <token>`
- `--protocol <xmpp>`
- `--verbosity <error|warn|chat|debug>`

## Publish

```bash
npm publish --workspace packages/streaming-chat-webclient-cli --access public
```
